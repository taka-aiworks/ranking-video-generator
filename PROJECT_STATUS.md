### **採用戦略: セミ自動化 = 効率性 × 品質管理**

#### **フロー設計（詳細版）**
```
1. キーワード入力（5秒）
   ↓
2. ChatGPT API分析・コンテンツ生成（30秒）
   ↓
3. 📝【重要】スクリプト編集画面（1-5分）
   - タイトル編集
   - ランキング項目調整
   - 説明文カスタマイズ
   - 時間配分調整
   ↓
4. 動画時間自動計算（コンテンツ量ベース、5秒）
   ↓
5. 動画生成・出力（30-180秒）
```

#### **セミ自動化の強み**
- **品質確保**: AIの効率 + 人間の最終調整
- **差別化**: 同じキーワードでも独自コンテンツ
- **実用性**: プロが実際に使える品質レベル
- **収益性**: 月20-45万円の現実的達成

#### **編集可能項目（実装予定）**
- **動画タイトル**: SEO最適化の手動調整
- **ランキング項目**: 商品名・価格・説明の修正・順序変更
- **時間配分**: 各項目の表示時間カスタマイズ
- **視覚設定**: 色・フォント・レイアウト微調整
- **内容追加**: 独自の項目・コメント追加

#### **動画時間計算ロジック**
```javascript
const calculateVideoDuration = (content, format, userAdjustments) => {
  const itemCount = content.items.length;
  const textLength = content.totalTextLength;
  const baseTime = 3; // 導入・まとめ
  
  if (format === 'short') {
    const autoTime = Math.min(itemCount * 12 + baseTime, 60);
    return applyUserAdjustments(autoTime, userAdjustments); // 最大60秒
  } else {
    const autoTime = itemCount * 30 + baseTime * 20;
    return applyUserAdjustments(autoTime, userAdjustments); // ミディアム用
  }
```
